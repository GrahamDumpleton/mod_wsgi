============
Contributing
============

mod_wsgi is maintained by its original author. See
:doc:`project-status` for the current maintenance posture.

Monetary support
----------------

If you would like to support the project financially, sponsorship
is set up through GitHub Sponsors:

* https://github.com/sponsors/GrahamDumpleton

Code contributions
------------------

Pull requests on the GitHub repository are welcome. If you have
fixed something for yourself or built a small feature you would
like to share, please feel free to send it through.

There is no obligation to contribute code in order to ask for
something, though. If you have hit a problem or have a suggestion,
opening a GitHub issue that describes it is just as helpful — and
often more so. mod_wsgi sits across both the Apache and CPython C
APIs, which makes the code a fairly specialised place to work, and
most fixes or small features are quicker for the author to develop
than they would be for a first-time contributor. If writing the
code yourself would be more friction than help, just describe the
problem or idea in an issue and the author will work from there.

* https://github.com/GrahamDumpleton/mod_wsgi/issues
